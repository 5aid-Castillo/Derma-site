Para montar el proyecto se deben colocar los archivos en la carpeta de "www" en el respectivo servidor local en el que se esté probando.

Dicho servidor necesita tener PHP 8.0.8 y MySQL para poder correr el proyecto.

CARPETAS:
---------------------

api:
   En la carpeta de api vienen los archivos de dicha api que se conectan tanto a la tienda, como al panel de administrador para realizar diversas acciones.
panel:
   En la carpeta de panel vienen los archivos tanto de frontend como de backend de la sección del panel de administrador.
img:
   En la carpeta de img vienen las imágenes usadas para el frontend de la tienda.
css:
   En la carpeta de css vienen todos los estilos que se usan para las diferentes secciónes de la página
js:
   En la carpeta de js vienen los archivos javascript que añaden funcionalidad principalmente a la sección de la tienda y checkout.
mail:
   En la carpeta de mail vienen los archivos necesarios para poder realizar el envío de correos mediante PHP, así como la plantilla usada.

----------------------
tiendaenlinea.sql:

    Este archivo contiene la configuración de la BD usada.

db.php:

    Este archivo es el que se usa como conexión a la base de datos.
